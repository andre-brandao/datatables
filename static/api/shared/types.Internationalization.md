
Used to define the text displayed in showcase components.


## Example

```ts
const i18n = {
    search: 'Search...',
    show: 'Show', 
    entries: 'lines',
    filter: 'Filter',
    rowCount: 'Showing {start} to {end} of {total} entries',
    noRows: 'No entries found',
    previous: 'Previous', 
    next: 'Next'
}
```
